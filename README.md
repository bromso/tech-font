## File formats
Following file formats will be included within this template when released:
* Sketch file
* Installable font (.eot, .ttf, .woff)
* SVG files
* SVG sprite file
* PNG files (Sizes: 32x32, 48x48, 64x64, 128x128, 256x256, 512x512)
* Documentation (README.md)

## Overall Symbol Structure
```
Symbol/
└── Technologies/
    ├── Cool tech logo #1
    │   ├── Flat
    │   ├── Outline
    │   └── Regular
    ├── Cool tech logo #2
    └── Cool tech logo #3
```

```
Symbol/
└── Technologies/
    └── Cool tech logo #1
        ├── Slice of group below with exports
        └── Group
            ├── (Aa) Graphical asset
            ├── (/) Graphical asset
            └── ([]) Graphical asset
```

## Overall Shared Styles
```
Shared Styles/
└── Technologies/
    ├── Flat border inside
    ├── Flat border center
    ├── Flat border outside
    └── Flat fill
```
